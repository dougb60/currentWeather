## 🚀 Instalando CurrentWeather

Para instalar o CurrentWeather, siga estas etapas:

git:
```
git clone https://github.com/dougb60/currentWeather.git
```

Linux e macOS:
```
yarn install;
yarn ios ou yarn android;
```

Windows:
```
yarn install
yarn ios ou yarn android;
```
